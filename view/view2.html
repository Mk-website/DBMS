Creating Views

Here's how you can create a view in MySQL:

Syntax

CREATE VIEW view_name AS SELECT column1, column2, ... FROM table_name WHERE condition;

Query

CREATE VIEW failStudents AS SELECT * FROM student WHERE marks < 180;
